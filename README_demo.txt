Shallow Water Demo by Stephen Thompson
======================================

This is a demonstration of solving the shallow water equations in real
time using GPU technology.

Note: This demo requires DirectX 10 compatible graphics hardware. 


Instructions
============

* Run shallow_water.exe to start the demo.
* Use WASD keys and PageUp/PageDown to move camera
* Right-click and drag to rotate camera
* Left-clicking will add water to the simulation (but see also
  "Left_Mouse_Action" at the right-hand side)
* Click the buttons under "Presets" to see various predefined
  scenarios.
* Visit http://www.solarflare.org.uk/shallow_water_demo/settings_guide.html
  for a full description of all the GUI settings.

For further details please refer to
http://www.solarflare.org.uk/shallow_water_demo/index.html.
